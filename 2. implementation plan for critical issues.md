```javascript
    company: companyName,
    companyIQScore: score,
    rating: getRating(score), // "STRONG" / "MODERATE" / "CAUTION" / "HIGH RISK"

    generatedAt: new Date().toISOString(),
    cta: "Get the full Standard Report for ₹499 →"
  };
}
```

**Cost per free score:** ₹31–41 (25–33 steps)
**Purpose:** Lead generation + viral sharing ("My stock just scored 82!"). Every free score user sees the paid report CTA.

**Why it's feasible for hackathon:** It's literally a subset of the Financial Agent we're already building. No extra development — just a lightweight endpoint that runs fewer TinyFish calls.

**How we manage abuse (free scores draining money):**
- Rate limit: 5 free scores per IP per day (enough for personal use, blocks scrapers)
- Edge caching: If a company was scored in the last 24h, serve cached score (costs ₹0)
- Soft wall: After 3 scores, prompt for email signup (builds mailing list)
